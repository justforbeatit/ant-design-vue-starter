The recommended package manager is [pnpm](https://pnpm.io), which is more excellent than npm and yarn

> Install packages:
```zsh
  pnpm i
```

> Start dev server:
```zsh
  pnpm dev
```

> Build for production:
```zsh
  pnpm build
```
